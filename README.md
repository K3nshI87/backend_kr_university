# Бэкенд курсовая
---
 Тема - Веб-сервис анализа и прогнозирования погоды

 Используемые технологии: Python, FastAPI, VsCode, API, PostgreSQL, Redis, наличие: внешнего вида страниц, соответствующего современным стандартам веб разработки, использование паттерна проектирования (MVC). 

 ---
 Функционал:
 1) Главная страница
 - Поиск своего города
 - Выбор города для отслеживания
 - Кнопки выбора за сколько дней показывать погоду (Сегодня, Завтра, 3 дня, Неделя)
 - Кнопка перехода на погоду на карте? (Если есть бесплатный API для рф)
 - Кнопка перехода в Личный кабинет для сохранения местоположения и выбора отображения 
 (темп C/F)
 (Скорость м/c или км/ч или миль/с)
 (давление мм.рт.ст/кПа)
 2) Личный кабинет
 3) Погода на карте

 ---
 Идея: разделить код на 3 части:
MVC pattern
Model — данные и работа с БД
View — отображение (HTML / frontend)
Controller — логика обработки запросов
